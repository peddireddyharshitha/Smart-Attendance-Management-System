# AI-Powered Smart Attendance Management and Certificate Eligibility Prediction System Using Zoom Data

## Overview

This project automates attendance analysis using Zoom attendance reports and predicts certificate eligibility using Machine Learning. It handles multiple join and leave sessions correctly by summing attendance durations and provides accurate attendance statistics.

## Features

* Zoom attendance data processing
* Multiple session handling
* Attendance percentage calculation
* Certificate eligibility prediction
* Exploratory Data Analysis (EDA)
* Logistic Regression model
* Model evaluation
* Model saving using Joblib

## Technologies Used

* Python
* Pandas

* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Machine Learning Algorithm

* Logistic Regression
